Hello world....

I thought it was appropriate to start off with the cheesiest, most overused phrase when it comes to web development but there's nothing wrong with adding a bit of cheesyness to your coursework, right?

This project is part of the coursework within the degree course for BSc(Hons) Web Design and Development at Edinburgh Napier University, where I currently study.

The task:

The task we have been given is to create a website that offers great functionality and shows off classical ciphers that we will create and code and implement using Javascript.

My website consists of;
  • an index.html page(the one you are looking at right now)
  • A design page which will outline relevant design decisions taken.
  • ROT 13 - Caesar Cipher
  • Mirror Writing Cipher
  • Atbash Cipher
There is of course a js folder which contains the javascript file needed for the ciphers, a fonts folder which contains the local files of the custom font declared in the css file which is also within its own folder, in order to follow the 'standard' structure for websites.
